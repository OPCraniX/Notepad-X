Notepad-X Help

Overview
Notepad-X is a tabbed desktop text editor for Windows focused on fast editing, code navigation, compare/preview workflows, shared notes, safer saves, and responsive handling of large files.

Quick Start
- `Ctrl+W` opens a file.
- `Ctrl+T` opens a new tab.
- `Ctrl+S` saves the current tab.
- `Ctrl+F` opens Find.
- `Ctrl+Q` opens compare mode.
- `Ctrl+Shift+P` opens Markdown Preview.
- `Ctrl+Shift+K` opens the command panel.
- `Ctrl+Shift+X` closes compare, preview, Find, Replace, Help, or About before exiting the app.

Main Window
- The center editor shows the active tab.
- The optional right pane shows compare content or Markdown preview.
- The optional far-right sidebar shows `Currently Editing` details for shared files.
- The status bar shows line/column, character counts, zoom, mode, CPU, memory, and clock.
- The line-number gutter can also show fold controls.
- The minimap shows a compressed overview of the current file.

Tabs
- Every file opens in its own tab.
- Tabs remember caret position and scroll position.
- Reopening the same file selects the existing tab instead of duplicating it.
- Tabs can be reordered with drag and drop.
- Right-click a tab for save, copy name/path, reveal, and move-to-new-window actions.
- Modified tabs show `*`.
- Session restore reopens file-backed tabs on launch.

Line Numbers and Folding
- `Settings > Numbered Lines` shows or hides the gutter.
- Clicking a line number copies that full line to the clipboard.
- Foldable sections show a boxed `+` or `-` in the gutter.
- Clicking the fold box collapses or expands that section.
- `F9` toggles the nearest foldable section containing the caret.
- `Shift+F9` collapses all folds.
- `Ctrl+F9` expands all folds.

Editing Helpers
- Auto-indent carries indentation to the next line.
- Extra indentation is added after `(`, `[`, `{`, or `:`.
- Auto-pair can insert matching quotes and brackets.
- Backspace removes paired characters when appropriate.
- Matching bracket highlight marks the nearest matching pair.
- These helpers work in the main editor and editable compare pane.

Autocomplete and Symbols
- Autocomplete is enabled by default for normal editable tabs.
- Suggestions can come from:
  current-file symbols
  words already in the file
  other open tabs
  syntax keywords
  Python builtins and keywords
  nearby project/module names
- `Ctrl+Shift+O` opens symbol navigation for the current file.
- `Ctrl+Alt+P` opens project symbol navigation.

Diagnostics
- Diagnostics highlight warnings and errors in normal editable files.
- Current warning types include:
  trailing whitespace
  long lines over 140 characters
- Current syntax/error checks include:
  Python syntax errors
  JSON parse errors
  XML parse errors
- Log and crash files can also surface traceback/error lines as diagnostics.
- Hover the mouse over a highlighted warning/error line to see the issue bubble.
- `Settings > Diagnostics` toggles diagnostics on or off.

Find, Replace, and Find In Files
- Find highlights matches live while you type.
- Replace uses the same live highlighting.
- `F3` moves forward through matches.
- `Shift+F3` moves backward.
- `Find In Files` searches inside a chosen folder and opens a results window.
- Search history suggestions appear in popup lists for Find and Find In.
- Search follows the editor pane clicked most recently while comparing.

Compare Mode
- `Ctrl+Q` opens compare mode.
- The source tab stays on the left and the selected comparison tab opens on the right.
- Compare mode uses the main window instead of a separate compare app.
- Normal text tabs remain editable in both panes.
- `Settings > Compare Multi-Edit` can mirror edits between both panes while comparing.
- The right pane has its own status readout and minimap.
- `Ctrl+Shift+X` closes compare mode.

Markdown Preview
- `Ctrl+Shift+P` opens live Markdown Preview in the right pane.
- Preview and compare share the same right-side workspace.
- Headings, lists, quotes, rules, fenced code blocks, emphasis, inline code, and links are rendered.
- `Settings > Sync PgUp/PgDn in Compare/Preview` keeps page navigation moving together between panes.

Minimap and Breadcrumbs
- `Settings > Minimap` shows or hides the minimap.
- Click or drag in the minimap to jump through the document.
- Diagnostic markers also appear in the minimap.
- `Settings > Breadcrumbs` shows file/location context above the editor.
- Breadcrumbs can include the nearest symbol while editing code.
- Clicking the breadcrumb filename copies just the file name; clicking the location copies the full path.

Command Panel
- `Ctrl+Shift+K` opens the docked command panel at the bottom of the app.
- It supports built-in commands and one-shot shell commands.
- Type `:` to open the built-in command autocomplete dropdown.
- Use `Up` / `Down` to move through suggestions and `Tab` to accept one.
- The built-in commands cover the `File`, `Edit`, `View`, `Settings`, and `Help` menus plus editor tools.
- Built-in commands include:
  `:help`
  `:symbols`
  `:project-symbols`
  `:fold`
  `:fold-all`
  `:unfold-all`
  `:lint`
  `:preview`
  `:compare`
  `:remote user@host:/absolute/path`
  `:autosave on|off`
  `:minimap on|off`
  `:diagnostics on|off`
- `cls` or `clear` clears the command output.
- `:help` prints the full built-in command list.
- Unknown `:` commands stay inside the built-in command system instead of running in the shell.
- Drag the top grip to resize the command panel vertically.
- The command panel is a quick runner, not a full interactive terminal emulator.

Remote Files
- `File > Open Remote (SSH)` or `Ctrl+Alt+O` opens a remote file.
- Use the format:
  `user@host:/absolute/path/to/file`
- Notepad-X fetches the file with `scp`, edits a local shadow copy, and pushes it back on save.
- Remote open/save is supported, but full remote project browsing is not.

Shared Notes
- Notes attach to selected text from the context menu.
- Notes support colors, replies, unread tracking, and export.
- `F3` jumps unread notes when Find and Replace are closed.
- `F4` cycles notes using the current note filter.
- Clicking a note marker opens its note popup.

Currently Editing
- `Ctrl+Shift+C` toggles the `Currently Editing` sidebar.
- It shows tracked editors for the active saved file.
- This sidebar can stay visible beside compare mode or preview.

Spell Check
- Spell Check underlines misspelled words in supported text-oriented modes.
- Right-click can show replacement suggestions and `Add to Dictionary`.
- `F7` toggles Spell Check.
- `Settings > Spell Check` also controls it.
- Spell Check stays out of preview tabs, virtual large-file tabs, and most code modes.

Themes, Syntax, and Language
- `View > Syntax Theme` switches themes.
- `Create Theme` builds custom themes.
- `View > Syntax Mode` overrides automatic syntax detection for the current tab.
- `Edit > Language` changes the visible UI language.
- Arabic and Hebrew can switch the UI flow to right-to-left automatically.

Save and Run
- `Ctrl+Shift+R` saves the current tab and launches supported script or HTML files.
- Supported targets include Python, JavaScript, PHP, batch, PowerShell, shell, and HTML.
- HTML opens in the default browser.

Encrypted Files
- `Save As Encrypted` writes a passphrase-protected `.npxe` copy.
- Opening an encrypted file prompts for the passphrase.
- Normal Save and Save As stay plain text.

Autosave, Backups, Session Restore, and Recovery
- `Settings > Auto Save` toggles autosave.
- Manual saves use atomic replace behavior.
- Backup snapshots are created on save.
- Session restore remembers open files, major editor toggles, and the main window size/state.
- Crash recovery can restore untitled work and modified tabs after a crash.
- Save conflict checks warn before overwriting a newer disk version.

Settings Menu
- `Settings` collects the checkmark/toggle options that used to be split across `Edit` and `View`.
- It includes Status Bar, Numbered Lines, Autocomplete, Spell Check, Auto Pair, Compare Multi-Edit, Minimap, Breadcrumbs, Diagnostics, Auto Save, Word Wrap, Markdown Preview, Sync PgUp/PgDn, Sound, and `Edit with Notepad-X`.

Large Files
- Large files can load in the background.
- Extremely large text files can open in virtual mode.
- Virtual mode keeps navigation available while limiting direct editing.
- Binary-like files open in preview-style mode instead of normal editing mode.

File Menu
- Open: Open a file.
- Open Project: Open a file plus related files from the same folder.
- Open Remote (SSH): Open a remote file through `scp`.
- Grab Git: Download a public GitHub repository and open chosen files.
- Recent: Reopen recent files.
- New Tab: Open a new untitled tab.
- Close Tab: Close the current tab.
- Save / Save All / Save As / Save Copy As: Standard save workflows.
- Save As Encrypted: Save an encrypted `.npxe` copy.
- Save and Run: Save and launch the current file.
- Print: Print the current saved file through the Windows print action.
- Export Notes: Export notes as JSON or Markdown.
